# Telegram Notes Bot

Telegram-бот для ведения личных заметок на Rust.

## Возможности

- Создание, просмотр, удаление заметок
- Тегирование заметок
- Поиск по тексту и тегам
- Напоминания с фоновой проверкой
- Экспорт всех заметок в текстовый файл
- Многопользовательская поддержка (каждый пользователь видит только свои заметки)

## Быстрый старт

### 1. Получите токен бота

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot`
3. Следуйте инструкциям и скопируйте токен

### 2. Настройте окружение

Создайте файл `.env` в корне проекта:

```
TELOXIDE_TOKEN=ваш_токен_здесь
DATABASE_URL=sqlite:notes.db
```

### 3. Запустите бота

```bash
cargo run
```

## Команды

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие |
| `/help` | Список команд |
| `/add <название> \| <текст>` | Добавить заметку |
| `/list` | Список заметок |
| `/view <id>` | Просмотр заметки |
| `/delete <id>` | Удалить заметку |
| `/search <запрос>` | Поиск заметок |
| `/tag <id> <тег>` | Добавить тег к заметке |
| `/bytag <тег>` | Заметки по тегу |
| `/remind <id> <дата>` | Установить напоминание |
| `/export` | Экспорт всех заметок |

Просто отправьте любой текст — он сохранится как заметка.

## Структура проекта

```
src/
├── main.rs      — Инициализация бота, диспетчер, проверка напоминаний
├── db.rs        — Операции с базой данных SQLite
├── handlers.rs  — Обработчики команд Telegram
├── models.rs    — Структуры данных (Note, Reminder)
└── utils.rs     — Вспомогательные функции
```

## Технологии

- **Rust** — язык программирования
- **Teloxide 0.17** — фреймворк для Telegram-ботов
- **SQLx 0.8** — асинхронный драйвер для SQLite
- **Tokio** — асинхронный рантайм
- **Chrono** — работа с датой и временем
